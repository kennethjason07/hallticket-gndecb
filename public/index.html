How to Use

1. Upload your Student Data Excel file.

2. (Optional) Upload your College Logo.

3. Verify/Edit Department and Exam details.

4. Choose Subject Selection mode (Excel or Manual).

5. Click Generate to download hall tickets.

Excel Format Requirements

The Excel file must have the following columns:

  • USN: Student USN
  • Name: Student Name
  • Subjects Applied: Comma-separated subject codes (if using Excel mode)
  • Example: 21CS51, 21CS52, 21CS53
